Projects

Graphical Reddit Opinion Miner

The Graphical Reddit Opinion Mining project uses Natural Language Processing (NLP) to analyze and visualize sentiments in Reddit posts through graphical representations. By employing the Natural Language Toolkit (NLTK) and Scikit-learn, the project performs tasks such as tokenization, classification, and semantic analysis to process text data. Reddit posts are retrieved using the Reddit API with a bearer token, and a classification model predicts the sentiment for user-specified phrases or keywords. The results are plotted on a graph, offering a clear visualization of the sentiment trends.

Sleep Disorder Detector

The Sleep Disorder Detector predicts Obstructive Sleep Apnea (OSA) by analyzing user-supplied records, combining smart watch data and questionnaire responses. This project offers early detection, helping users seek timely medical intervention. Due to the COVID-19 outbreak, many people acquired health monitoring devices like blood pressure monitors and oxygen saturation monitors to track their health at home. Post-pandemic, these devices often remain idle. This research proposes repurposing these devices to gather valuable data for predictive health analytics, developing models for conditions like OSA.

NutriCompass

The NutriCompass project leverages Google's Gemini Pro AI to analyze meal images and provide detailed nutritional insights. Built with Streamlit, the application allows users to upload a meal image, which is processed by Gemini AI to determine calorie content and a breakdown of key nutrients. The app integrates the Python Imaging Library (PIL) for image handling, ensuring seamless processing. Users can view the uploaded image and initiate analysis through an interactive interface. Results are presented in a user-friendly format, helping individuals make informed dietary decisions by understanding the nutritional profile of their meals.

Automated Amazon Price Tracker

The Automated Amazon Price Tracker monitors daily product prices and sends email alerts when they drop below a set threshold. This project offers cost savings, convenience, and customization, allowing you to manage your budget effectively. It provides hands-on experience with Python and web scraping, and can be scaled to include other platforms and features

Hazardous NEO Prediction

This project predicts hazardous Near-Earth Objects (NEOs) using NASA’s dataset of over 300,000 records. Features such as a NEO’s relative velocity, diameter, and distance from Earth were analyzed using models like random forest, logistic regression, and neural networks. The random forest achieved 95% accuracy and a ROC-AUC score of 99%, showcasing machine learning's potential in addressing space safety challenges.

Automated Rain Alert

This project offers timely weather updates, helping users plan their day effectively. It provides practical experience with API integration and automation. The app helps us stay prepared and informed with its user-friendly weather alert system.
